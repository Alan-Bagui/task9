# Task9 - Panel AdminLTE con Laravel

## Descripcion
Panel de administracion desarrollado con AdminLTE 3 integrado a Laravel.
Contiene las secciones de la Actividad 11 adaptadas a un panel admin.

## Secciones
- Inicio / Dashboard
- Fotos
- Contacto

## Tecnologias
- Laravel
- AdminLTE 3
- Bootstrap 4
- Blade Templates

## Como ejecutar
1. Clonar el repositorio
2. Ejecutar `composer install`
3. Copiar `.env.example` a `.env`
4. Ejecutar `php artisan key:generate`
5. Ejecutar `php artisan serve`
